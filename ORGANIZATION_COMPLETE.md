## 🚀 Quick Start with New Structure

```bash
# Documentation index
cat docs/README.md

# Complete guide

cd docs/
```

### Run Scripts
```bash
# Start server
python scripts/start_server.py

# Quick analysis
python scripts/quick_analysis.py

# List all scripts
ls scripts/
```

### Check Configuration
```bash
# View configs
ls config/
```
